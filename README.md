# ATL-project
A chat bot made with python for ATL project  
Contributors:   
Ankith  

# Requirements - 
- python 3.7 or newer  
- git  
# Installation steps:
```bash
git clone "https://github.com/AnkithAbhayan/ATL-project" "chat-bot"/
cd "chat-bot"/
python -m pip install -U requirements.txt
```
Installation done!  
  
# How to run?
```bash
cd "chat-bot"/
python chat-bot.py
```
